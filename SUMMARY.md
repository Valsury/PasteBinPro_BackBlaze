## 🆓 Бесплатная конфигурация

### Компоненты (все бесплатно):

1. **Render** - хостинг приложения
   - 750 часов/месяц бесплатно
   - Автоматический SSL
   - Автодеплой из Git

2. **Backblaze B2** - хранилище файлов
   - 10 GB бесплатно навсегда
   - 1 GB трафика/день
   - S3-совместимый API

3. **OpenRouter** - LLM API
   - Бесплатные модели без пополнения
   - mistral-7b, gemini-flash, llama-3
   - Неограниченные запросы

4. **Render PostgreSQL** - база данных
   - Бесплатно 90 дней
   - Потом $7/месяц
   - Альтернатива: Supabase (1 GB бесплатно навсегда)

### Итоговая стоимость:
- **Первые 90 дней**: $0/месяц
- **После 90 дней**: $7/месяц (только PostgreSQL)
- **Полностью бесплатно**: $0/месяц (с Supabase вместо Render PostgreSQL)

## 🔑 Необходимые переменные окружения

### Обязательные для Render:

```bash
# OpenRouter (бесплатные модели)
LLM_PROVIDER=openrouter
LLM_API_KEY=sk-or-v1-xxxxx
LLM_MODEL=mistralai/mistral-7b-instruct

# Backblaze B2 (10 GB бесплатно)
STORAGE_TYPE=s3
S3_ENDPOINT=s3.us-west-004.backblazeb2.com
S3_ACCESS_KEY=<ваш-keyID>
S3_SECRET_KEY=<ваш-applicationKey>
S3_BUCKET_NAME=pastebin-pastes
S3_SECURE=true
S3_REGION=us-west-004

# URL приложения
APP_URL=https://your-app.onrender.com
```

### Автоматические (Render установит сам):
```bash
DATABASE_URL=<render-postgresql-url>
PORT=<render-port>
SECRET_KEY=<auto-generated>
```
