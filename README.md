**PasteBin Pro** - это современная веб-платформа для быстрого создания, обмена и хранения кода и текста с интегрированным AI-помощником. Приложение сочетает в себе элегантный дизайн в стиле Together.ai, мощные возможности генерации контента и удобный интерфейс для профессионалов.

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/yourusername/pastebin-pro.git
cd pastebin-pro
```

### 2. Установка зависимостей
```bash
# Создание виртуального окружения
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.venv\Scripts\Activate.ps1  # Windows PowerShell

# Установка зависимостей
pip install -r requirements.txt
```

### 3. Настройка окружения
Создайте файл `.env` со следующими настройками:
```bash
# Flask
FLASK_APP=app.py
FLASK_ENV=development
SECRET_KEY=your-secret-key-here

# PostgreSQL
DATABASE_URL=postgresql://username:password@localhost:5432/pastebin_db

# Backblaze B2 (опционально, для облачного хранилища)
STORAGE_TYPE=s3
S3_ENDPOINT=s3.us-west-004.backblazeb2.com
S3_ACCESS_KEY=your-access-key
S3_SECRET_KEY=your-secret-key
S3_BUCKET_NAME=pastebin-pastes
S3_REGION=us-west-004

# OpenRouter AI API (опционально)
LLM_PROVIDER=openrouter
LLM_API_KEY=sk-or-v1-your-api-key
LLM_MODEL=mistralai/mistral-7b-instruct
```

### 4. Запуск базы данных
```bash
# С использованием Docker Compose
docker-compose up -d

# Или вручную установите PostgreSQL
```

### 5. Инициализация базы данных
```bash
# Создание таблиц
python -c "from app import app, db; app.app_context().push(); db.create_all()"

# Применение миграций
alembic upgrade head
```

### 6. Запуск приложения
```bash
python app.py
```

Приложение будет доступно по адресу: `http://localhost:5000`

## 🎯 Использование

### Создание пасты
1. Перейдите на страницу **"Создать"**
2. Введите название и содержимое
3. Выберите категорию (язык программирования или тип контента)
4. Установите время жизни пасты с помощью интерактивного ползунка
5. Нажмите **"Создать пасту"**

### AI-помощник
1. Откройте страницу **"AI-помощник"**
2. Выберите **категорию текста** (креативный, бизнес, код и т.д.)
3. Настройте **стиль/тип** и дополнительные параметры
4. Введите описание того, что хотите сгенерировать
5. Нажмите **"Создать текст"**
6. Используйте результат: скопируйте или создайте пасту

### Система авторизации
- **Регистрация**: создайте учетную запись с email и паролем
- **Вход**: используйте email или username для входа
- **Профиль**: просматривайте статистику и управляйте аккаунтом

### Модели базы данных
- **User**: пользователи системы
- **Paste**: пасты (основная сущность)
- **Tag**: теги для организации паст
- **AppStats**: статистика приложения

### API Endpoints
- `GET /` - Главная страница
- `GET /create`, `POST /create` - Создание пасты
- `GET /paste/<id>` - Просмотр пасты
- `GET /recent` - Недавние пасты
- `GET /login`, `POST /login` - Вход в систему
- `GET /register`, `POST /register` - Регистрация
- `GET /logout` - Выход из системы
- `GET /profile` - Профиль пользователя
- `GET /ai` - AI-помощник
- `POST /ai/generate-text` - Генерация текста

## 🔧 Администрирование

### Управление миграциями
```bash
# Показать статус миграций
alembic current

# Создать новую миграцию
alembic revision -m "Описание изменений"

# Применить миграции
alembic upgrade head

# Откатить миграцию
alembic downgrade -1
```

### Управление хранилищем
- **Локальное хранилище**: данные сохраняются локально
- **Backblaze B2**: 10 GB бесплатного облачного хранилища
- **AWS S3**: совместимо с Amazon S3 API
- **MinIO**: локальный S3-совместимый сервер

### Мониторинг
- **Статистика**: количество паст, активных паст, просмотров
- **Логи**: подробное логирование операций
- **Очистка**: автоматическое удаление истекших паст

## 🚀 Развертывание

### Бесплатный деплой на Render (рекомендуется)
1. **Получите API ключи**:
   - [Backblaze B2](https://www.backblaze.com/b2/sign-up.html) (10 GB бесплатно)
   - [OpenRouter](https://openrouter.ai/) (бесплатные AI модели)

2. **Загрузите код в GitHub**

3. **Создайте Blueprint на Render**:
   - Откройте [Render Dashboard](https://dashboard.render.com/)
   - Нажмите **New → Blueprint**
   - Подключите ваш GitHub репозиторий

4. **Настройте переменные окружения**:
   ```bash
   # OpenRouter API
   LLM_API_KEY=sk-or-v1-xxxxx
   LLM_MODEL=mistralai/mistral-7b-instruct
   
   # Backblaze B2
   S3_ENDPOINT=s3.us-west-004.backblazeb2.com
   S3_ACCESS_KEY=your-access-key
   S3_SECRET_KEY=your-secret-key
   S3_BUCKET_NAME=pastebin-pastes
   S3_REGION=us-west-004
   ```

5. **Дождитесь деплоя** (3-5 минут)

### Продакшен настройки

# Используйте gunicorn для продакшена
gunicorn -w 4 -b 0.0.0.0:$PORT app:app

## 🐛 Устранение неполадок

### Общие проблемы
- **PostgreSQL не подключен**: убедитесь, что база данных запущена
- **Миграции не применяются**: проверьте Alembic конфигурацию
- **Хранилище недоступно**: проверьте сетевые настройки и ключи доступа
- **AI API не работает**: проверьте API ключ и подключение к интернету

### Логирование
- Проверьте логи Flask приложения
- Используйте `print()` для отладки
- Настройте уровни логирования через переменные окружения
