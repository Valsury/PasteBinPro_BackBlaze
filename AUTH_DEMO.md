# 🔐 Система авторизации (Demo)

## Что добавлено

Добавлена визуальная система авторизации и регистрации в режиме "для вида" (без реальной функциональности).

## Новые страницы

### 1. 🔑 Страница входа (`/login`)
- Красивая форма входа с полями email/username и пароль
- Кнопка "показать/скрыть пароль"
- Чекбокс "Запомнить меня"
- Ссылка на восстановление пароля
- Ссылка на регистрацию
- При отправке формы показывается уведомление "Демо-режим"

**Файл**: `templates/login.html`

### 2. 📝 Страница регистрации (`/register`)
- Форма с полями:
  - Имя пользователя (username)
  - Email
  - Пароль
  - Подтверждение пароля
- Индикатор силы пароля (слабый/средний/хороший/отличный)
- Проверка совпадения паролей в реальном времени
- Чекбокс согласия с условиями использования
- Валидация на клиенте (JavaScript)
- При отправке формы показывается уведомление "Демо-режим"

**Файл**: `templates/register.html`

### 3. 👤 Страница профиля (`/profile`)
- Аватар пользователя
- Статистика (количество паст, просмотров, дней с регистрации)
- Табы:
  - **Мои пасты** - список паст пользователя (пусто)
  - **Приватные** - приватные пасты (пусто)
  - **Настройки** - настройки профиля (disabled)
- Кнопки "Настройки" и "Выход"

**Файл**: `templates/profile.html`

## Для полной реализации потребуется

1. **Flask-Login** - управление сессиями
   ```bash
   pip install Flask-Login
   ```

2. **Flask-Bcrypt** - хеширование паролей
   ```bash
   pip install Flask-Bcrypt
   ```

3. **Flask-WTF** - защита форм (CSRF)
   ```bash
   pip install Flask-WTF
   ```

4. **Обновить модель User** в `models.py`:
   - Добавить методы для работы с паролями
   - Добавить Flask-Login методы

5. **Middleware для проверки авторизации**:
   - Декораторы `@login_required`
   - Проверка прав доступа

6. **Связать пасты с пользователями**:
   - Использовать поле `author_id` в модели Paste
   - Фильтрация паст по пользователю

## Использование

1. Перейдите на страницу `/login` или `/register`
2. Заполните форму (данные можно вводить любые)
3. Нажмите кнопку отправки
4. Увидите уведомление о демо-режиме
